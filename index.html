Vercel Team Deployment

To share this tool with your team without each person needing an API key, deploy via Vercel:

1. Push this folder to GitHub
2. Go to vercel.com → New Project → Import from GitHub
3. Add Environment Variable:
   ANTHROPIC_API_KEY = sk-ant-...
4. Deploy → share the URL with team

The /api/generate.js function keeps the key server-side.

Once deployed on Vercel, team members open the URL and generate content — no key needed on their end.
